# Agent Personas
Before performing any task, the agent MUST declare which persona from `/.agent/config.md` they are manifesting for that task.
Every session should begin by reading `/.agent/worklog.md` to ensure continuity.
Refer to `/.agent/orchestration.md` for multi-agent coordination.
