# Premium Design Tokens & System

Manifest as the **Frontend Wizard**. Use these tokens to ensure consistent, high-end aesthetics across all components.

## 🎨 Color Palette (HSL)
- **Brand Indigo**: `239 84% 67%` (Primary)
- **Deep Slate**: `222 47% 11%` (Background/Text)
- **Soft Emerald**: `156 72% 38%` (Success/Pass)
- **Vibrant Amber**: `38 92% 50%` (Warning/Pending)
- **Electric Red**: `0 84% 60%` (Error/Plagiarism)

## ✨ Motion Presets
- **Fast In-Out**: `cubic-bezier(0.4, 0, 0.2, 1)` (300ms) - Hover/Toggle.
- **Spring Up**: `cubic-bezier(0.175, 0.885, 0.32, 1.275)` (500ms) - Modals/Cards.

## 🧊 Glassmorphism
- **Standard**: `backdrop-blur-xl bg-white/70 border border-white/20`
- **Dark**: `backdrop-blur-xl bg-slate-900/80 border border-slate-700/50`

## 📝 Typography
- **Heading**: `Inter`, weight `800` (ExtraBold) or `900` (Black).
- **Body**: `Inter`, weight `400` (Regular) or `500` (Medium).
- **Mono**: `JetBrains Mono` or `Monaco` for code.
